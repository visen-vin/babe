# Vaspbot PRD (Product Requirements Document)

## 1. Vision
Vaspbot is not just a chatbot; it's an **Agent Framework**. The goal is to create a highly autonomous assistant that can use tools, remember long-term context, and eventually interact with web environments (browser automation/hijacking).

## 2. Core Features (Milestones)
- **M1: The Brain:** Connectivity with Gemini via LangChain.
- **M2: Memory:** Implementation of chat history and file-based context.
- **M3: Tools:** Giving the bot "hands" (Search, Calculator, File System).
- **M4: Autonomy:** Turning the bot into an Agent that makes decisions.
- **M5: Hijack Mode:** Browser automation to bypass standard API limits.

## 3. Tech Stack
- **Languages:** Node.js.
- **Core Framework:** LangChain.
- **LLM Engine:** Google Gemini.
- **Automation:** Playwright.
